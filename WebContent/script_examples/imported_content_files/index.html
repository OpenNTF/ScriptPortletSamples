About this sample

This sample shows the use of imported content: an image, a JSON data file, and an HTML fragment. All are loaded via Ajax calls. Each of these files is referenced with a relative link in the index.html page, and each relative link is replaced by a WCM Element tag when imported. The WCM element name is the same as the relative pathname. The JSON file and the HTML fragment are loaded by the application JavaScript code, and the URLs are read from the hidden "a" tags that are in the index.html file.

Note: The Javascript that loads the HTML fragment and the JSON file may not run standalone (from the file system) in some browsers due to security restriction.
