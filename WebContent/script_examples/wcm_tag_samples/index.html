About this sample

This sample shows some of the WCM tags that you can use within HTML elements for a Script Portlet. You can use WCM tags because a Script Portlet is rendered using the WCM Rendering Portlet. Note the for the LDAP attributes, the exact LDAP attributes available will depend on your LDAP configuration, so some of the attributes shown here may not always be present.
